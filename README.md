# Books Dataset - Data Visualization Project

## Project Overview

This project focuses on Data Visualization using a Books Dataset collected through web scraping.

The objective is to transform raw data into meaningful visualizations and discover useful insights from book pricing and stock availability.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset Information

* Total Records: 1000
* Total Columns: 6

Columns:

* Title
* Title_URL
* Image
* Price
* instock
* btn

## Visualizations Created

### 1. Top 10 Most Expensive Books

A horizontal bar chart showing the ten highest-priced books.

### 2. Price Distribution

A histogram displaying the distribution of book prices.

### 3. Stock Availability Distribution

A pie chart showing stock availability proportions.

### 4. Top 20 Books Price Comparison

A bar chart comparing the prices of the twenty most expensive books.

## Key Insights

* Book prices vary significantly across the dataset.
* Most books fall within a moderate price range.
* A small number of books are premium-priced.
* Stock availability information helps understand inventory status.

## Conclusion

The project successfully transformed raw book data into meaningful visualizations. The charts provide insights into pricing patterns and stock availability, helping users better understand the dataset.
